# Hello Bolg

基于[vuepress](https://vuepress.vuejs.org/)([中文链接](https://vuepress.vuejs.org/zh/))搭建的个人博客

在 vuepress 的基础上再进行了一层封装，并且增加了交互式体验，更加易用

安装命令

```sh
npm i -g qblog-cli
```

查看版本号

```sh
qblog-cli --version
```

使用方法

```sh
qblog-cli create
```

输入 create 命令后进入创建项目流程，第一步输入项目名，第二步输入你的博客项目 git 地址
